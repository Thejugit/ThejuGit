# 🎯 Quick Start Guide

## ⚡ 3 Minute Setup

### Step 1: Edit Your Profile (2 min)
Open `README.md` and find-replace:
- `John Doe` → Your name
- `YOUR_USERNAME` → Your GitHub username
- Update social links

### Step 2: Customize Languages & Tools (1 min)
Edit badges in the "💻 Languages and Tools" section

### Step 3: Done! 🎉
Push to GitHub and watch it auto-update!

---

## 🎨 Two Approaches

### **Approach A: Direct README Editing (Fastest)**
- Edit `README.md` directly
- Best for quick customization
- Changes are permanent
- → **Use this if you want to get started immediately**

### **Approach B: Config-Based (More Flexible)**
- Edit `config.json` with your info
- Run: `python scripts/generate_from_config.py`
- Regenerate README anytime
- → **Use this if you plan to regenerate often**

---

## 📝 What to Update

### Must Update:
- [ ] `John Doe` → Your name
- [ ] `YOUR_USERNAME` → GitHub username
- [ ] Social links (Twitter, LinkedIn, Email)
- [ ] Project links & descriptions

### Should Update:
- [ ] Languages and Tools (match your stack)
- [ ] Current Projects section
- [ ] Bio/description

### Optional:
- [ ] YouTube API setup (auto-update videos)
- [ ] Change colors/badges

## 🚀 After Setup

Your README will automatically update:
- ✅ GitHub stats (daily)
- ✅ Top languages (daily)
- ✅ YouTube videos (daily, if configured)
- ✅ Follower counts

No maintenance needed!
